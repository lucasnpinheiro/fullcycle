# BUILD

### Build image golang.

```bash

   docker build -t lucasnpinheiro/fullcycle:go .

```

### List image.

```bash

   docker images

```

### Run image.

```bash

   docker run lucasnpinheiro/fullcycle:go

```

### Pull image.

```bash

   docker pull lucasnpinheiro/fullcycle:go

```

### Push image.

```bash

   docker push lucasnpinheiro/fullcycle:go

```
